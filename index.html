政策 · 监管 — Rules of the Game

合规成为生存底线
04·01 · 强监管

广电新规落地

国家广电总局明确：未履行备案程序的存量动画微短剧 全网强制下线，新作必须"先备案后上线"。

  • AIGC 类、漫画类首次纳入
  • 重点监管：版权、未成年人保护、内容红线
  • 对小团队、野蛮生长模式打击最大
红果 · 主动治理

下架 1718 部、核查 1.5 万部

一季度下架违规 AI 漫剧 1718 部，专项核查 1.5 万部，额外处置 670 部。4·7 起进一步收严审核标准。

  • 未授权 IP → 下架
  • AI 素材违规 → 下架
  • 低质解说 / 擦边 → 下架
版权 · 原创保护

国家知识产权局出面"立规"

版权是平台最大隐性成本。对 QQ 而言，绑定阅文 / 腾讯视频 IP 是天然安全港。

平台政策 · 转向

从"保底躺赚"到"精品分账"

红果"流量灌溉"→ "价值深耕"。全行业转向精品化——对中小 CP 是重大利空、对优质 CP 是利好。

09

QQ 漫剧策略参考 — What It Means for Us

基于 0327 / 0417 两份内部汇报 + 行业情报交叉推导

核心定位：不做第二个红果，不做第二个火龙。依托 IM 生态、游戏协同、AI 玩法，走"社交化漫剧"的差异化路径。5 亿 MAU 天然流量 × 35 岁以下年轻用户与漫剧受众高度重合，是 QQ 的核心资产。

01

行业从"保底躺赚"转向"精品分账"

EVIDENCE红果、火龙双双收紧低质内容的保底；广电新规强制备案。
ACTIONQQ 初版政策"无保底 + 最高 100% 分成 + 全网 100 万播放量优先"与行业趋势完全对齐。建议在初期就强化"优质内容评级（S+/S/A/B）"的运营动作，把"精品优先"做成 QQ 的品牌标签。
02

游戏 × 漫剧是 2026 最重要融合方向

EVIDENCE《气运三角洲》29 小时破 2 亿；和平精英、王者荣耀主动找短剧 / 漫剧合作；37 互娱、米哈游也在接触 CP 做定制系列。腾讯长青游戏（三角洲、王者、金铲铲、DNF）天然具备漫剧化基础。
ACTION这是 QQ 真正的"护城河"，整个腾讯生态里 QQ 是最有资格做游戏漫剧落地场景的。
  • 抢 IP 首发权：与天美 / 琳琅协同，新游 / 新赛季配套漫剧首发放 QQ
  • "边看边玩"闭环：漫剧播放页"立即体验"→ 跳转游戏下载 / 小游戏
  • 剧情分支 × 游戏奖励：互动漫剧选择 → 解锁游戏道具 / 皮肤
03

女频是最大增量蓝海，小红书是自来水主阵地

EVIDENCE女频仅占 10–20%，但 #甜宠# 累计播放超 28 亿；小红书女频短剧口碑反哺抖音 / 红果；《宇宙尽头的书店》《大小姐》等精品化女频开始跑出来。
ACTION
  • 女频差异化供给：QQ 用户年轻化 + 女性比例，主动在 CP 合作中扶持女频赛道
  • 小红书反哺：做"女频漫剧种草 → QQ 观看"的流量链路
  • 脑洞秀 UGC 女频：邀请好友虚拟形象出演甜宠剧情，天然适配女频社交传播
04

AI 工具链已标准化，成本不是壁垒

EVIDENCEWorkRally、Seedance、腾讯云 / 商汤 / 万兴工具已非常成熟，爆款《西游》10 人团队 20 天 10 万成本就能做出来。
ACTIONQQ 不需要自建生产工具，应该聚焦 分发 × 社交 × 互动 的差异化能力。把脑洞秀升级为"UGC 漫剧创作 + 社交平台"——这是行业其他玩家做不了的事情。
05

流量规模是 CP 合作的硬门槛

EVIDENCE点众原话"给什么内容，关键看你们能给多少流量"——100w DAU 达不到他们的阈值。
ACTION路线 B（动态页小规模尝试）对 CP 吸引力过低，只能拿到阅文二三线内容，用户体验也难撑起循环。强烈建议走路线 A（底 Tab + 公众号 + 动态页 + PC，目标千万级 DAU），这是与行业头部玩家对话的最低门槛。
06

时间窗口在 2026 Q3-Q4 关闭

EVIDENCE红果 MAU 3 亿、火龙已上线、百度柚漫剧筹备中、小红书 / 快手 / 视频号 / 阿里都在入局。字节主导但未形成绝对垄断。
ACTION5 月 MVP → 6 月底灰度 → Q3 全量 → Q4 高速增长 的时间表非常关键。延迟半年，百度 / 阿里可能已经圈走一批 CP 和用户心智。现在就是最好的入场时机。
07

出海是后手但值得预留接口

EVIDENCE海外 2025 规模 40 亿美元，ReelShort 北美单日流水破百万美金。井漫已 1000 部 / 月。
ACTION国内 MVP 优先，但 B 端工具和内容合约设计时预留海外分发权的权益条款，后续可通过 WeSing / QQ 国际版或新产品接入海外 IP。

给 Neo 的 3 个决策建议

  1. 坚定走路线 A：路线 B 在当前行业阶段对 CP 吸引力不足，且会错失 2026 窗口期。
  2. 游戏漫剧是差异化第一王牌：立即启动与天美 / 琳琅的三角洲 / 王者 IP 漫剧联动，这是 QQ 唯一能在字节 / 百度 / 阿里面前打出的独占牌。
  3. 女频 + UGC 脑洞秀 双引擎：作为社交化差异的长期护城河。小红书联动 + 脑洞秀换脸二创，能把"看漫剧"升级为"玩漫剧"。
